# ChatGPT Share Bundle

This is a lightweight snapshot of the current NAS website for discussing the ZenOps subdomain update.

## Included

- `screenshots/home.jpg` - visual reference for the home page
- `screenshots/zenops.jpg` - visual reference for the ZenOps page
- `code/index.html` - home page markup
- `code/products/index.html` - products overview markup
- `code/products/zenops/index.html` - ZenOps page markup
- `code/assets/site.css` - shared site styles
- `code/assets/site.js` - shared site scripts

## Notes

- Source snapshot came from `projects/nas-website/NotAloneStudios_public_clean.zip`.
- Heavy assets were omitted on purpose to keep the bundle small.
- The current code still references ZenOps as internal site paths such as `/products/zenops/` and `/contact/#zenops`.
